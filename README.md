# mqtt-server



### Persistent MQTT session using mqtt.js library






##### Start Publisher
`yarn nodemon src/pub.ts`


##### Start Subscriber 1
`yarn nodemon src/sub1.ts`


##### Start Subscriber 2
`yarn nodemon src/sub2.ts`
